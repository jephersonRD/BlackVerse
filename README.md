## ¿Qué es Blackverse?

Plataforma de GPU cloud donde puedes rentar máquinas virtuales con GPUs potentes (RTX 4090, RTX 5080, A6000, etc.) por horas o días. Sin necesidad de comprar hardware costoso.

### Uso común:
- 🎮 **Cloud Gaming** — Juega en cualquier dispositivo
- 🤖 **IA & ML** — Entrena modelos sin GPU propia
- 🎬 **Rendering** — Renderizado rápido de proyectos
- 📺 **Streaming** — Stream con calidad máxima
